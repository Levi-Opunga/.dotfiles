# Dotfiles

This as repository for my dot files

## Dependencies

- gnu-stow
- i3
- picom
- polybar
- neovim
- zsh
- Kitty/Alacrity
- powerlevel10k
- lf
- rofi

## To use clone the repository into the home directory for your user

```bash
stow .

```

> <span style="color: red">If directories or files already exist gnu-stow throws an error</span>
